# Заголовок проекта

Десктопное приложение для языковой школы "Леарн" (Learn)

## Начало работы

1. Открыть исполняемый файл (ярлык/в папке программы).
2. Зарегистрироваться как администратор (код) или просматривать информацию как пользователь.

### Необходимые условия
Необходимые условия:
* Windows 10/11.
* Intel Pentium 2 или выше.
* 512 мб оперативной памяти.
* 1гб свободного места на диске.

### Установка

Установка:
1. Перейти по ссылке https://langlearn.ru.
2. Выбрать нужную версию установщика (Например, для Windows 10/11 - win64.exe) и установить.
3. Запустить установщик и следовать инструкциям.

Завершите пример получением некоторых данных о системе или использования их для небольшой демонстрации
